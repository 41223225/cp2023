如何繪製國旗

1首先給予圖形們必要工具

#include <stdio.h>用於輸入和輸出的函數 (列印文字等等)  
#include <gd.h>  繪圖工具
#include <math.h>  計算工具

繪製此檔，並以img的圖片格式匯出

void draw_圖名_flag(gdImagePtr img);

記住必須先給予圖形的最外框才能創建圖片(可選擇給予數字或者比例)
int main() {
    // 給予國旗邊框
    int width = XXX;
    int height = (int)(width * X / Y);

然後給予創建彩色圖片，且不允許圖像中的像素是透明或半透明(0為不允許1為允許)

(將繪製圖形出圖的形式)

gdImagePtr img = gdImageCreateTrueColor(width, height); gdImageAlphaBlending(img, 0);

此段話為把圖檔存在哪裡，並且嘗試打開，如打開失敗將會輸入一個非零的值(錯誤碼)

FILE *outputFile = fopen("./../images(前面為將圖檔存在哪裡)/圖名_flag.png", "wb");

if (outputFile == NULL) {

fprintf(stderr, "Error opening the output file.\n");

return 1;

}

此段話為將圖片輸出為img檔，而這裡的9指的是將檔案壓縮最大程度(0~9) output消除之前的file(並刷新數據)gdImageDestroy(img);同上，只是此為刷新輸出圖片

return 0 會將這個整數的數值傳回給作業系統, 在這個程式結束成功結束後。

gdImagePngEx(img, outputFile, 9); fclose(outputFile); gdImageDestroy(img); return 0;

2給予圖形們常用的工具

此為打開這個gd程式庫並且開始給予所需參數
void draw_國旗_flag(gdImagePtr img) {
    int width = gdImageSX(img);
    int height = gdImageSY(img);
      這裡是給予你所需要的顏色(後面三排數字分別為RAD 紅，綠，藍  顏色亮度，最高亮度為255)
    int blue = gdImageColorAllocate(img, 0, 0, 255);
    int white = gdImageColorAllocate(img, 255, 255, 255);
    int red = gdImageColorAllocate(img, 255, 0, 0);
